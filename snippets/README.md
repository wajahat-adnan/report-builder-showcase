# Selected Code Excerpts

## Overview

This directory is reserved for small, self-contained code excerpts that demonstrate coding style without exposing the production implementation.

The complete Report Builder source code remains in a private repository.

## What May Be Published

Suitable public excerpts may include:

- Generic data models
- Small validation helpers
- Standalone formatting utilities
- Simplified interface components
- Synthetic examples that do not reveal production logic
- Educational versions of non-proprietary patterns

## What Must Stay Private

Do not publish:

- Complete OCR extraction logic
- Template-alignment implementation
- Anchor and parsing rules
- Clinical measurement thresholds
- Full report-generation logic
- Production prompts
- Private fixtures or benchmark datasets
- Builder or reconstruction scripts
- Code copied from the private repository if it reveals the full workflow

## Publication Standard

Every excerpt should be:

- Small enough to review quickly
- Understandable without the private codebase
- Free of secrets and personal information
- Based on synthetic examples
- Clearly labeled as illustrative
- Non-reconstructive

## Review Checklist

Before committing an excerpt, confirm that it contains no:

- API keys
- Passwords
- Tokens
- Private email addresses
- Patient information
- Local file paths
- Proprietary thresholds
- Internal repository references
- Hidden dependencies on private modules

No production source excerpts have been published yet.
